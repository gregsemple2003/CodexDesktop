# Problem 0002 Option C Plan 0001

## Summary

Create a separate Q-and-A appendix for each pass update.

## What Changes

- each pass update ends with a short question-answer table
- unresolved questions are listed separately
- direct answers live outside the main prose

## Files Or Artifact Types

- pass update template
- handoff or status-note template

## Rollout

1. Add a Q-and-A appendix section to pass-update templates.
2. Teach writers to fill it before sending long summaries.
3. Trial it on a few active tasks.

## Success Checks

- users can scan a single answer table fast
- unresolved questions are obvious
- the extra appendix does not duplicate most of the reply
